Model Namespace Reference

Detailed Description

Setup the Hamiltonian and corresponding values.

This namespace contains model parameters, such as the coupling (or nearest neighbor exchange energy) \(J\), and the transverse field strength \(g\). By default the model describes the quantum Ising model with a transverse field:

\[ H = \frac{1}{2}\sum_{n=1}^L J S^z_n \otimes S^z_{n+1} + gS^x_{n} \]
