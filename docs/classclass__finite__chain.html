class_finite_chain Class Reference

Detailed Description

Storage class for the finite chain and effective environments during finite DMRG The storage is always partitioned into two lists, corresponding to the two halves about the current position. The back and front of these lists correspond to the current state of the superblock's left and right parts, respectively, while "MPS_C" is the central \( \Lambda^A \). Once full, the particle content of the chain should match the length given in the constant max_length.
