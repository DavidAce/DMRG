Math Namespace Reference

Detailed Description

Math functions.

Contains functions do math on more general types, print out vectors and do numerical integration on lambdas using GSL.

Examples of integration:

\( \int_0^1 x^2 dx = \)

compute_integral([](double x) { return x*x; }, {0,1})

\( \int_1^\infty x^{-2} dx = \)

compute_integral([](double x) { return 1/(x*x); }, {1,INFINITY})

\( \int_{-\infty}^\infty \exp(-x^2) dx = \)

compute_integral([](double x) { return std::exp(-x*x); }, {-INFINITY,INFINITY})
