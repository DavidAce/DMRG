template<typename T1, typename T2>
auto Math::mod(const T1 x, const T2 y)

MatLab-style modulo operator.

Parameters
x first number
y second number
Returns modulo of x and y. Example, mod(-0.5,10) = 9.5, instead of -0.5 as given by xy.

template<typename T1, typename T2>
std::vector<T2> Math::LinSpaced(T1 num, T2 min, T2 max)

MatLab-style linearly spaced array.

Parameters
num number of linearly spaced values
min minimum value in range
max maximum value in range
Returns std::vector<T2>. Example, Linspaced(5,1,5) gives a std::vector<int>: [1,2,3,4,5]

template<typename Input, typename From, typename To>
auto Math::prod(const Input& in, const From from, const To to)

Product operator for containers such as vector.

Parameters
in a vector, array or any 1D container with "<code> .data() </code>" method.
from first element to multiply
to last element to multiply
Returns std::vector<T2>. Example, let my_vector = {1,2,3,4}. Then prod(my_vector,0,3) = 24.

template<typename F>
double Math::compute_integral(F func, std::pair<double, double> const& range, double epsabs = 1.49e-8, double epsrel = 1.49e-8, int limit = 50)

Numerical integrations.

Examples of integration:

compute_integral([](double x) { return x*x; }, {0,1})

compute_integral([](double x) { return 1/(x*x); }, {1,INFINITY})

compute_integral([](double x) { return std::exp(-x*x); }, {-INFINITY,INFINITY})
