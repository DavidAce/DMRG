class_mpo Class Reference

Detailed Description

MPO representations of the Hamiltonian. Class for the Hamiltonian MPO, either as a rank-4 MPO used in DMRG, or as a time evolution operator \( \exp(-iH\delta t) \) as used in TEBD.
