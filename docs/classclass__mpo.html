class_mpo Class Reference

Detailed Description

Superblock Class

This class contains the Hamiltonian MPO, current wave function MPS, left and right environment blocks and routines to contract, diagonalize, truncate and update them.

◆ compute_F()

Tensor< std::complex< double >, 4 > class_mpo::compute_F ( double  a)

Returns the moment generating function MPO hamiltonian as a rank 4 MPO. F := exp(aM), where a is a small parameter and M is an MPO. 

    0       1
    |       |
    [exp(aH)]
    |       |
    2       3

Where H = H_even + H_odd, so exp(aH) can be Suzuki-Trotter decomposed just as in iTEBD
