class_mpo class

MPO representations of the Hamiltonian. Class for the Hamiltonian MPO, either as a rank-4 MPO used in DMRG, or as a time evolution operator as used in TEBD.

Textra::Tensor<std::complex<double>, 4> class_mpo::compute_F(double a)

Returns the moment generating function MPO hamiltonian as a rank 4 MPO. F := exp(aM), where a is a small parameter and M is an MPO.

    0       1
    |       |
    [exp(aH)]
    |       |
    2       3

Where H = H_even + H_odd, so exp(aH) can be Suzuki-Trotter decomposed just as in iTEBD

Textra::Tensor<std::complex<double>, 4> class_mpo::compute_G(double a)

Returns the characteristic function MPO hamiltonian as a rank 4 tensor. G := exp(iaM), where a is a small parameter and M is an MPO. 0 1 | | [exp(iaH)] | | 2 3
