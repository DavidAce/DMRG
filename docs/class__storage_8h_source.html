15  # Storage Class
16 
17  During the infinite-DMRG part of the algorithm, the current state for each chain-length
18  is stored in a `class_storage` object, for later use in the finite-DMRG stage.
19 
20  Specifically, this class stores objects from the current superblock:
21  - The left and right environment blocks `Lblock` and `Rblock`, both type `Textra::Tensor3`, and
22  - the MPS tensors \f$\Gamma^{A,B}=\f$ `MPS.GA, MPS.GB` and \f$\Lambda^{A,B}=\f$ `MPS.LA, MPS.LB`, required for computing \f$\Theta\f$, following
23  the notation used in [Frank Pollmann's notes](http://quantumtensor.pks.mpg.de/wp-content/uploads/2016/06/notes_1.pdf):
24  \f$\Theta= \Lambda^B\Gamma^A\Lambda^A\Gamma^B\Lambda^B\f$. <br> Here \f$\Gamma^{A,B}\f$ are a rank-3 tensors of type `Textra::Tensor3` and \f$\Lambda^{A,B}\f$
25  are rank-1 tensors of type `Textra::Tensor1`.
26 
27  All objects are stored and indexed by their position relative to the final chain of length `max_length`.
28 */
