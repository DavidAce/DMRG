class_storage Class Reference

Detailed Description

Storage Class

During the infinite-DMRG part of the algorithm, the current state for each chain-length is stored in a class_storage object, for later use in the finite-DMRG stage.

Specifically, this class stores objects from the current superblock:

  • The left and right environment blocks Lblock and Rblock, both type Textra::Tensor3, and
  • the MPS tensors \(\Gamma^{A,B}=\) MPS.GA, MPS.GB and \(\Lambda^{A,B}=\) MPS.LA, MPS.LB, required for computing \(\Theta\), following the notation used in Frank Pollmann's notes: \(\Theta= \Lambda^B\Gamma^A\Lambda^A\Gamma^B\Lambda^B\).
    Here \(\Gamma^{A,B}\) are a rank-3 tensors of type Textra::Tensor3 and \(\Lambda^{A,B}\) are rank-1 tensors of type Textra::Tensor1.

All objects are stored and indexed by their position relative to the final chain of length max_length.
