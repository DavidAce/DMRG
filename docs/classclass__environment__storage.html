class_environment_storage class

Storage class for environments during finite DMRG During the infinite-DMRG part of the algorithm, the current state for each chain-length is stored in a class_storage object, for later use in the finite-DMRG stage.

Specifically, this class stores objects from the current superblock:

  • The left and right environment blocks Lblock and Rblock, both type Tensor3, and
